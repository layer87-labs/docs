Open source tools and GitHub Actions for CI/CD workflows, maintained by [Layer87](https://layer87.de).

## Tools

### relctl

Release automation CLI for GitHub. Determines the next SemVer version from branch naming conventions, creates release drafts, uploads build assets, and publishes — from a single binary.

- [Documentation](/docs/relctl/intro)
- [GitHub](https://github.com/layer87-labs/relctl)
- [GitHub Action](https://github.com/layer87-labs/relctl-action)
