# relctl pr

Manage GitHub pull requests.

```bash
relctl pr [flags]
```

## pr info

Get pull request info for the current or specified PR.

```bash
relctl pr info [flags]
```

```
  -h, --help                      help for info
  -c, --merge-commit-sha string   merge commit SHA to look up
  -n, --number int                overwrite the PR number
  -v, --verbose                   verbose output
```
